## Codebase Patterns
- Use `bun:sqlite` with `Database` from `bun:sqlite` for all SQLite operations
- Always use `IF NOT EXISTS` in CREATE TABLE migrations
- All DB code lives in `src/db.ts`
- Use `db.prepare<ReturnType, ParamsType>()` for type-safe prepared statements; `stmt.get()` for single rows, `stmt.all()` for arrays
- Use `RETURNING` clause in INSERT/UPDATE to get the row back in one query; use a subquery inside `RETURNING` to include joined data (e.g. `categoryName`) without a second query
- For aggregations with LEFT JOIN + GROUP BY, use `COALESCE(SUM(...), 0)` to return 0 instead of NULL for categories/months with no transactions
- For date-range month filtering, compute `start` and `end` ISO dates in TypeScript and pass as parameters (same pattern as `getTransactionsByMonth`)
